  
<span style="font-family:Arial sans-serif;font-size:16px;">WD Evaluation period</span>

  
<span style="font-family:Arial sans-serif;font-size:14px;">This example explains how to limit the use of an application to a given period (evaluation period).</span>

<span style="font-family:Arial sans-serif;font-size:14px;">The following topics are presented in this example:</span>

<span style="font-family:Arial sans-serif;font-size:14px;"> 1/ the protection of an application for a given duration</span>

<span style="font-family:Arial sans-serif;font-size:14px;"> 2/ the management of the registry</span>

  
<span style="font-family:Arial sans-serif;font-size:14px;">Summary of the example supplied with WINDEV: </span>

<span style="font-family:Arial sans-serif;font-size:14px;">When this example is started for the first time, it is activated for an evaluation period set to 5 days.</span>

<span style="font-family:Arial sans-serif;font-size:14px;">The information regarding the date when it was first started is stored in the registry and a control key is used to check whether this date was not modified by the end user. At the end of the evaluation period, the application is locked, unless the end user provides the code for unlocking the application</span>
